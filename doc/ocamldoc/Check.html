type generator_arity =
| Ga_zeroary
| Ga_unary
| Ga_binary
| Ga_listary
| Ga_mary of int (*The generator associated to a constructor could have one of the following arity class:

  • Ga_zeroary, if the generator is associated to a constant or 0-ary constructor,
  • Ga_unary, if the generator is associated to a unary functional constructor,
  • Ga_binary, if the generator is associated to a binary functional constructor,
  • Ga_listary, if the generator is associated to a functional constructor with variable arity (i.e. a constructor that has a single argument which is a list of elements of the relational type at hand),
  • Ga_mary n, if the generator is associated to a constructor with an arbitrary number of arguments n. Note that for a generator with arity class Ga_mary n:
  • n can be 1 when the generator cannot be considered as having a variable arity,
  • n cannot be 0 or 2,
  • n can be greater than 3.
*)
